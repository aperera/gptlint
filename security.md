# Security Policy

Our team takes security bugs in `gptlint` very seriously. We appreciate your efforts to responsibly disclose your findings and will make every effort to acknowledge your contributions.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report any security vulnerabilities at [gptlint@teamduality.dev](mailto:gptlint@teamduality.dev).
